The HTML Boilerplate 4.2

Gives us the structure to HTML file.

  • <!DOCTYPE html> : Tells the cumpter what version of html this file was made on
  • <html lang="en">####</html> : everything is writen within this (lang) tells us what language of text content
  • <head>####</head> : Area where important info about the website is placed, that will not be displayed to user
    • <meta charset="UTF-8"> : character set encoding of page (this is a comman character set) other metas exsit, like view size stuff
    • <title>####</title> : Title of the page (displayed on tab bar)
  • <body>#####</body> : Main body for the HTML file

You can just reuse the same basic boilderplate over and overagain if you want with diff titles.
On VScode just do ! on first line and hit enter to insert boiler plate for html file

The List Element 3.0


Unordered List

Use <ul> </ul> to set up the list.

In between we use <li> </li> to set up each element within the list


For example :

<ul>
  <li>Cool</li>
  <li>Math</li>
  <li>Games</li>
</ul>

Will become
  • Cool
  • Math
  • Games

Ordered List

Use <ol> </ol> to set up the list.

In between we use <li> </li> to set up each element within the list


For example :

<ol>
  <li>Cool</li>
  <li>Math</li>
  <li>Games</li>
</ol>

Will become
  1. Cool
  2. Math
  3. Games

Nesting and identation 3.1

Example list within a list :

<ul>
  <li>Cool</li>
  <li>Math</li>
  <ol>
    <li>Addition</li>
    <li>Subraction</li>
    <li>Multiplication</li>
  </ol>
  <li>Games</li>
</ul>

We get
  • Cool
  • Math
    1. Addition
    2. Subtraction
    3. Multiplication
  • Games
Can use ordered and unordered lists for this

Anchor Elements 3.2

  <tag attribute=value anotherattribute=value >Content</tag> can have more than one attribute

Which gives This is a google link

We have more local and global attributes other than that within the following links
  1. Basic attributes

Image Elements 3.3

This is what the image element looks like

  <img src="url" alt="backup text" /> : Inserts an image

  • src : src tells the image element what the source of the image is
  • url : the location of the image
  • alt : gives alternative text description
  • This is also a void element

This image element holds a link to a photo of Lebron's game 7 block

  <img src="https://www.mercurynews.com/wp-content/uploads/2016/08/20160628__LEBRON-BLOCK1.jpg?w=1024" alt="Lebron game 7 block/>

Which produces this photo

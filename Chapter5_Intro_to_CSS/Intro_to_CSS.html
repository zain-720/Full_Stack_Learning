Chapter 5: Intro to CSS

How to add CSS 5.1


Three ways to add CSS into an HTML file

  • Inline: <tag style="css" />
  • Internal: <style>css</style>
  • External: <link href="style.css"/>

Inline

Goes into the same line as a particular HMTL element

EX: On the HTML boilerplate
  • <html>style="background: blurlywood"</html>

style attribute takes in CSS code here to change the background color. Best for single elements. Does not get overwrote by a following external or internal CSS prompt

Internal

Goes into the head of a boilderplate EX (all within head section)

  • <style>html{ background: red; } </style>

All of the space within the style tag can be used for CSS. Useful for limiting style to the specific HTML file it is located within. Best for single HTML pages

External

Goes into the head of a boilderplate EX (all within head section)

  • <link rel="stylesheet" href="./styles.css" />

A the link element would access a CSS file named styles.css would contain the css rules (in this ex: html{ background: green;}). This lets css code be added onto multiple HMTL files faster. (rel defines the relationship to the file being linked to).

This is the most commonly used type of CSS coding

CSS Selectors 5.3


All Setups here are within the stylesheet style.css

Class Selector

(.red-heading{ color: ###}). This will set the color for a class to the color selected for the class.
Ran as <h2 class="red-heading">Something</h2>.
This would set the text in here to the color of the class.

This was made from green-heading

Paragraphs and h can have text changed by the class selector its not specific.

Id Selector

These are only to be applied to single elements. Setup as #main{color: ###}.
Ran as <h2 id="main">Something</h2>

This was made from m-mag

Attribute selector

Allow for particular attribute values to be affected only.
Example a setup for changing the color of draggables p[draggable]{ color: ### }
Now all draggables are some color.

Aqua draggable

Color for false draggables need to be set seperately.

Universal Selector

This will set everything to the color set. *{color: ###}

Notes

We can do more than change color with this, we can change font size, text alignment, etc.

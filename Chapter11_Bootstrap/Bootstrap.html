11.1 Bootstrap layout

Understanding the 12 column Bootstrap layout system:


Made of 3 components, a div with class container, inside that another div with class row,
and inside the row is where we have our items which will be laid out using the column system.


This is showcased in the example below.


Hello
Hi
Something

The "container" is automatically setup, different kinds of containers exist in bootstrap for different size displays.


"col" is also modifiable, with col-2, col-4 representing taking up 2 and 4 columns worth of space out of a total of 12 column spaces dividing the space.
Base col with try to equally take up whatever space is left


In this:
"Hello" is col-md-4 col-sm-2, so takes up 4 col at medium size and 2 at small,
"Hi" is col-md-6 col-sm-3
"Something" is col-md col-sm

Hi
Something

We also have breakpoints like sm, md etc which go in as col-sm-"wtv else" which set a break to the size of the column, helps for optimizing for specific devices
With this in mind its possible to input multiple breakpoints so it has an option for each size of device the screen is set to

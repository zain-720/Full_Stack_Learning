We can do a flexbox inside a part of a grid to get the best of both

10.2 Grid Placement

How to layout items in the grid(s) created: configure grid cells to create grid item layouts on the grid.


Lets say we have a grid made up of 3 items. Inside Flexbox centers the items, while otuside grid makes the layout


1
2
3

Using grid-column, grid-row spann # we can set the boxes to take more than 1 cell of space to create these layouts using the cells available


Other helpful tools:


gird-column-start, grid-column-end set which column a box will start and end at (this can be done for rows as well). -1 represents the rightmost/furthestdown box respectively.
while 1 represents the leftmost/highest box respectively. These tools are just more specific versions of grid-column and grid-row, grid-area combines both. (All of this can be found with Googles HTML inspect tools)

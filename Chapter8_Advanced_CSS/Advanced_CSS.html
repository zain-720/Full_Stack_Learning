    put. @media (max-width: 600px){} Then inside we write css for screens blow or equal to 600px wide in it.

    8.3 Media Queries

    Recall @media (max-width: 600px){} :
    That is saying that at that if the diplay width is less than or qual to 600px then the styling within it should be used
    Ex :
    @media (max-width: 600px){ h1{ font-size: 15px;}} , so set font size of h1 to 15px when screen width is 600px or less

    The following box below will change size depending on the display width from its based 300px height and width to 200px width and height


    @media (min-width: 600px){} :
    This will do the opposite of max-width, now anything 600px in width or greater gets its changes.

    The box below will go from 300px h,w to 200px h,w when the screen size reaches 600px or greater


    We can combine the min and max to make ranges of effect @media (min-width: 600px) and (max-width: 900px){}
    Can also make a statement which effects anything equal to, less than, or greater than the range given @media (max-width: 600px) and (min-width: 900px){}

7.1 Combining CSS Selectors

More specific styling per class by combining them
Ex:

<p>Yellow Text</p>
<div class="box inner_box">
  <p>White Text</p>
</div>

Combining this with stlye sheet modifications of:

p {
  color: yellow;
}
.inner-box p {
  color: white;
}

These settings create: yellow text on the outside and white text in the inner box

Combining selectors (methods)

Group rule : (Selectors under the same rule set by placing them together with commas)

selector, selector {
  color: blue;
}

Parent child selector rule : (Direct child of parent class (selector) with the given name (selectorC, p, etc) will have these effects)

selector > selectorC {
  color: blue;
}

Decendant rule : (as long as selectorC is in someway a decendant of selector, it will be affected by it)

selector selectorC {
  color: blue;
}

Chaining : (apply given styles where all the selectors listed are true)
example : for <h1 id="title" class="big heading">Hello World</h1> to stylize this specificially chain as,
h1#title.big.heading (selectors are chained so it applies to this specific type of element, (element should always be first in naming ex (li, p, etc)))

selectorselectorC {
  color: blue;
}

These rules are also able to be combined together ex decendant and chanining together

7.2 CSS Positioning

4 different positioning values. Static, Relative, Absolute, Fixed



Static : this is the HTML default. Everything just flows downwards as its added in.



Relative : Where an item is position relitave to where its default position would put it in this case 50px from left, 50px from top.





Absolute : Position relative to nearest positioned ancestor or top left corner of webpage. Nearest ancestor here is box2 (This time parent div is set to box so its outline is confined)
Z-axis operations determine which boxes are ontop of each other ex here box2 has z index 10 and absolute box has z index of -1, so it is put behind box 2 (absolute is always top unless its -1)











Fixed : Relative to the top corner of the browser widnow (that is where the small green box comes from)

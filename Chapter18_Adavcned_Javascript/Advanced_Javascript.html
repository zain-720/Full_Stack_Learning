18.2 Passing Functions as Arguments

Its possibel to pass a function name into a function as an argument then use it later to call said function.
function bob(opp, num1, num2) { return opp(num1, num2) } , This is a possible function in javascript.
These functions that take other functions as inputs are called "Higher Order Functions"

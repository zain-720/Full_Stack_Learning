23.0 What is Node.js


Node.js is a Java based backend framework, framworks are filled with prebuilt components that make back end much simplier than writing without it on the regular language.


Node lets run javascript on our own local computer rather than leaving it locked to the webpage. its asynchronous event-driven. Node.js lets us run stuff on a server (computer). It basically lets us write javascript for a backend as well.

23.3 Node Package Manager NPM

Where to find npm packages


The NPM is a place which collects modules people have made for Node.


Work with npm in 23.3 folder index.js


Initialize first NPM project:


  1. At wanted location in terminal write : npm init
  2. Answer the given questions (leave some blank for now, or put -y at the end of init to skip this, or just npm i)
  3. Once all of this is done a new file called package.json is created

Now we can import npm packages to our .js file


In the terminal where npm was initialized we can do (npm i sillyname) to add packages to use. Once ran a new dependency is added to our package.json. It also created a folder called node_modules which is filled with the codes for package (sillyname)


Reading the documentation for the added package will tell us how to use it. We use the new package in the index.js in folder 23.3


Can also just import keyword instead to bring in the npm and node packages (as seen in index.js). However to allow this to be used with node we need to go into the package.json and add a feild tpye: and set it to "moudle"

    36.9 : Map/Filter/Reduce

    Going to go through the .map(function) function in greater detail and look at some of the related functions that help deal with arrays such as .map
    .filter
    .reduce
    .find
    .findIndex


    output = array.map(func) -Create a new array (if we choose to capture the output) by doing something with each item in an array (using the func given)


    output = array.filter(func) -Create a new array by keeping items that return true (everytime the func returns true, the index that got the boolean true return is added to the output)


    output = array.reduce(func) -Accumulate a value by doing something to each item in an array (sum up / combine the items in array given).
    Function should include an variable for accumulation and current number (func(acum, curr)). reduce will make sure these varables are set correctly, so long as the output from the func in the acumulated value.


    output = array.find(func), finds first instance of an item in the array. Ex : (return num > 10) is the func given, then output is the first value in the array where the value is greater than 10


    output = array.findIndex(func), finds first instance of an item in the array and returns the index for it. Ex : (return num > 10) is the func given, then output is the index of the first value in the array where the value is greater than 10

    How to create interactive React apps, to do this we need to understand State.


    State : Our UI is a function of the state, state changes brought about by the UI cause page changes

    Imperative doesnt use variables but instead directly changes the properties of the element


    Declarative uses need to be rerendered in order for the change in the Ex: style property, to be read and acted on. this takes the use of React Hooks which lets us actively change these state variables for our app.
